# ChatStruct

ChatStruct is a web application that organizes WhatsApp group messages into structured categories such as:

- Announcements
- Study Materials
- Doubts
- Important Messages
- Upcoming Events

Features:
• WhatsApp Web integration  
• AI message categorization using Gemini  
• Smart dashboard UI  
• Upcoming events extraction  
• Real-time message updates

Tech Stack:
Frontend: React + Vite  
Backend: Node.js + Express  
WhatsApp Integration: whatsapp-web.js  
AI: Gemini API
